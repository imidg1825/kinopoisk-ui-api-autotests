Дипломная работа студента группы 111.2 IND
Тема проекта

Автоматизация тестирования веб-приложения Кинопоиск (UI + API).

Платформа выполнения работы

Дипломная работа выполнена на следующей платформе:

Операционная система: Windows

Браузер: Google Chrome

Язык программирования: Python 3.14

Среда разработки: Visual Studio Code

Используемые технологии и инструменты

В рамках выполнения дипломного проекта были установлены и использованы следующие инструменты:

pytest — фреймворк для автотестов

selenium — автоматизация UI тестирования

requests — тестирование API

webdriver-manager — автоматическая установка ChromeDriver

allure-pytest — формирование отчётов

flake8 — статический анализ кода

Установка зависимостей

Перед запуском проекта необходимо установить зависимости:

pip install -r requirements.txt

Запуск тестов
Запуск всех тестов (UI + API)

pytest -vv

Запуск только API тестов

pytest -m api -vv

Запуск только UI тестов

pytest -m ui -vv

Формирование отчёта Allure
Генерация результатов

pytest --alluredir=allure-results

Просмотр отчёта

allure serve allure-results

После выполнения команды автоматически открывается браузер с отчётом.

Проверка кода на соответствие стандартам

Проект проверен инструментом статического анализа flake8.

Запуск проверки:

flake8 .

Ошибок стиля и нарушений PEP8 в проекте не выявлено.

Конфигурация flake8 вынесена в файл .flake8.

Архитектура проекта

Проект реализован с использованием:

Page Object Model (для UI тестов)

pytest fixtures

разделения тестов через маркеры (api, ui)

централизованной конфигурации

Цель дипломной работы

Целью дипломной работы является разработка и реализация автоматизированного тестового проекта для проверки функциональности веб-приложения Кинопоиск с использованием современных инструментов автоматизации тестирования.
